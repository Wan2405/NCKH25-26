# NCKH25-26 — Hệ thống tự động kiểm thử và gỡ lỗi mã nguồn Java sinh bởi mô hình ngôn ngữ lớn

Automated Testing and Debugging System for Java Code Generated by Large Language Models.

## Tính năng chính / Features

| Thành phần | Mô tả |
|---|---|
| **Java Executor** | Biên dịch và chạy mã nguồn Java tự động — Compile & run Java source code |
| **Bug Analyzer** | Phân tích lỗi biên dịch và lỗi runtime, đề xuất cách sửa — Parse errors & suggest fixes |
| **Test Generator** | Sinh test case JUnit 5 tự động cho các phương thức — Auto-generate JUnit 5 tests |
| **LLM Client** | Tích hợp API OpenAI để phân tích và sửa lỗi nâng cao — OpenAI-compatible API |
| **Pipeline** | Quy trình tự động: biên dịch → chạy → kiểm thử → phân tích → sửa lỗi |
| **REST API** | API FastAPI để tương tác với hệ thống — FastAPI endpoints |

## Cài đặt / Installation

```bash
# Yêu cầu: Python 3.10+, JDK 17+
pip install -r requirements.txt
```

## Chạy ứng dụng / Running

```bash
# Khởi động API server
uvicorn src.api:app --reload --host 0.0.0.0 --port 8000
```

Mở trình duyệt tại `http://localhost:8000/docs` để xem API documentation.

## API Endpoints

| Method | Path | Mô tả |
|---|---|---|
| `GET` | `/health` | Kiểm tra trạng thái server |
| `POST` | `/compile` | Biên dịch mã Java |
| `POST` | `/run` | Biên dịch và chạy mã Java |
| `POST` | `/analyze` | Phân tích lỗi và đề xuất sửa |
| `POST` | `/generate-tests` | Sinh test case tự động |
| `POST` | `/pipeline` | Chạy toàn bộ quy trình kiểm thử |

### Ví dụ sử dụng / Example Usage

```bash
# Biên dịch và chạy mã Java
curl -X POST http://localhost:8000/run \
  -H "Content-Type: application/json" \
  -d '{"source_code": "public class Hello { public static void main(String[] args) { System.out.println(\"Hello!\"); } }"}'

# Phân tích lỗi
curl -X POST http://localhost:8000/analyze \
  -H "Content-Type: application/json" \
  -d '{"error_output": "Main.java:3: error: ; expected", "error_type": "compilation"}'

# Chạy pipeline đầy đủ
curl -X POST http://localhost:8000/pipeline \
  -H "Content-Type: application/json" \
  -d '{"source_code": "public class Test { public static void main(String[] args) { System.out.println(\"hi\"); } }", "expected_output": "hi"}'
```

## Chạy test / Running Tests

```bash
python -m pytest tests/ -v
```

## Cấu hình LLM (tùy chọn)

Để sử dụng tính năng phân tích bằng LLM, cấu hình biến môi trường:

```bash
export OPENAI_API_KEY="your-api-key"
export OPENAI_API_BASE="https://api.openai.com/v1"  # hoặc API tương thích khác
export OPENAI_MODEL="gpt-3.5-turbo"
```

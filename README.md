# Hệ thống Tự động Kiểm thử và Gỡ lỗi Mã nguồn Java sinh bởi Mô hình Ngôn ngữ Lớn
## Automated Testing & Debugging System for LLM-Generated Java Code

> **NCKH 2025–2026** — Nghiên cứu Khoa học / Scientific Research Project

---

## Tổng quan / Overview

Hệ thống này tự động hóa toàn bộ vòng lặp kiểm thử – gỡ lỗi cho mã nguồn Java được tạo ra bởi các mô hình ngôn ngữ lớn (LLM).

This system automates the full test–debug loop for Java source code generated by large language models:

1. **Biên dịch / Compile** — Biên dịch mã Java và thu thập tất cả lỗi cú pháp.
2. **Thực thi / Run** — Chạy chương trình và ghi lại stdout, stderr, mã thoát.
3. **Phân tích lỗi / Analyze** — Phân tích lỗi biên dịch và lỗi runtime thành báo cáo có cấu trúc kèm gợi ý.
4. **Tự động sửa / Auto-fix** — Gửi mô tả lỗi cho LLM và nhận lại mã đã sửa, lặp lại tối đa N lần.
5. **Sinh kiểm thử / Generate tests** — Tạo lớp kiểm thử JUnit 5 (dựa trên template hoặc LLM).
6. **REST API** — Phục vụ toàn bộ pipeline thông qua các endpoint HTTP.

## Cài đặt / Configuration

Tạo file `.env` hoặc xuất các biến môi trường:

```bash
# LLM (tuỳ chọn – bỏ qua để chạy không có AI)
export OPENAI_API_KEY="sk-..."
export LLM_MODEL="gpt-4o-mini"          # default
export LLM_BASE_URL="https://api.openai.com/v1"  # thay để dùng API tương thích khác

# Java
export JAVAC_PATH="javac"
export JAVA_PATH="java"

# Pipeline
export MAX_FIX_ITERATIONS="3"
export EXECUTION_TIMEOUT="15"

# API server
export API_HOST="0.0.0.0"
export API_PORT="8000"
```

## Sử dụng / Usage

### 1. Khởi động API server

```bash
python main.py serve
```

Swagger UI: http://localhost:8000/docs

### 2. Chạy pipeline qua CLI

```bash
python main.py run examples/BuggyStringUtils.java
```

### 3. Sinh test JUnit 5 qua CLI

```bash
python main.py test examples/Calculator.java
```

## API Endpoints

| Method | Path | Mô tả |
|--------|------|--------|
| GET  | `/health` | Kiểm tra trạng thái server |
| POST | `/compile` | Biên dịch mã Java, trả về lỗi cú pháp |
| POST | `/run` | Biên dịch và chạy mã Java |
| POST | `/pipeline` | Chạy toàn bộ pipeline (compile → run → analyze → fix → test) |
| POST | `/generate-tests` | Sinh lớp kiểm thử JUnit 5 |
| POST | `/explain` | Giải thích mã bằng LLM |

### Ví dụ / Example

```bash
curl -X POST http://localhost:8000/pipeline \
  -H "Content-Type: application/json" \
  -d '{
    "source_code": "public class Hello { public static void main(String[] args) { System.out.println(\"Hi\"); } }",
    "max_iterations": 3,
    "generate_unit_tests": true
  }'
```

## Kiểm thử / Testing

```bash
pytest tests/ -v
```

43 bài kiểm thử bao phủ:
- Biên dịch thành công và thất bại
- Chạy chương trình, stdin, ngoại lệ runtime
- Phân tích lỗi biên dịch và lỗi runtime
- Sinh test JUnit 5
- Pipeline đầy đủ
- Tất cả các API endpoint

## Luồng hoạt động / Workflow

```
Mã Java (LLM-generated)
        │
        ▼
  ┌─────────────┐
  │   Compile   │◄──────────────────────────────┐
  └──────┬──────┘                                │
         │ Lỗi biên dịch                         │
         │         Thành công                    │
         ▼                                       │
  ┌─────────────┐                                │
  │     Run     │                                │
  └──────┬──────┘                                │
         │ Lỗi runtime                           │
         │         Thành công                    │
         ▼                                       │ LLM fix
  ┌─────────────┐       ┌──────────────┐         │
  │   Analyze   │──────►│  LLM Client  │─────────┘
  └─────────────┘       └──────────────┘
         │ Không còn lỗi
         ▼
  ┌─────────────────┐
  │ Generate Tests  │
  └─────────────────┘
```
